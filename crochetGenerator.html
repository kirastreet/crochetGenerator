Generate your own crochet patterns and discover new forms. Experiment with materials, colors, and scales.

For more info on this project, visit the project website here.

PATTERN MODE:
Randomize - Each round has a random stitch
Normal - Each round increases normally using double crochet stitches

CONSTRUCTION MODE:
Rounds - work in the round
Rows - coming soon, working in a zig zag in straight stitches

Number of initial stitches indicated the number of the stitches in the first round or row. The pattern can start with a chain circle or magic circle, whichever you're more comfortable with. Minimum of 4, maximum of 10.

Number of rows or rounds indicates how many repeated rows or rounds you have. Minimum of 3, maximum of 10.

Generate - If you change anything, click the Generate button to generate a new pattern. You can also keep clicking it with the same settings to get a different pattern with the same settings.

Word Pattern - For the Randomize mode, you can see a basic word pattern showing how many of what kinds of stitches are in each row or round.

Save a picture of the pattern to your computer with the Save button. It will save as crochetpatternXX.png.

TIP:
Think of these patterns as a guide. You don't have to follow them exactly if you don't want to. Change the stitches or the number of stitches in each row or round. Make the pattern your own.
